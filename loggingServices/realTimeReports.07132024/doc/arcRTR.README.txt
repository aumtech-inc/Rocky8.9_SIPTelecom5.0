- goFindFile()
	- search $LOG and $LOG/load/loaded.<date>, returns fullPath 
	  and doesItExist
	  if ( notFound) 
	  {
		sets doesItExist to 0
	  }
	  else
	  {
		sets fullPath
		sets .doesItExist to 1
	  }
	  return(0);
